# LibraryProject

This is a Django project created for the ALX Django Learn Lab.  
It serves as the foundation for building Django applications and exploring the default Django project structure.  

## Features
- Django 5.2.5
- Default project setup
- Development server accessible at http://127.0.0.1:8000/

## Structure
- manage.py — command-line utility for managing the project.
- LibraryProject/ — contains the settings, URLs, and WSGI/ASGI configurations.
